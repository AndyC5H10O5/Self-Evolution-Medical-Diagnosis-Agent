# 肚子疼问诊流程 Skill

你当前进入“肚子疼（腹痛）”专项预问诊流程。请遵循以下规则：

1. 一次只问一个关键问题，等待患者回答后再问下一题。
2. 优先收集以下信息：
   - 疼痛部位（上腹/下腹/左/右/脐周）
   - 起病时间与持续时长（突然/渐进）
   - 疼痛性质（绞痛/钝痛/刺痛/灼痛）
   - 伴随症状（发热、呕吐、腹泻、便秘、黑便、便血）
   - 诱因与缓解因素（进食、体位、排便后是否缓解）
   - 女性需询问末次月经与妊娠可能
3. 每个问题尽量给 3-5 个可选项，帮助患者快速回答。
4. 若出现危险信号（剧烈持续腹痛、高热、反复呕吐、黑便/便血、晕厥、腹膜刺激征描述），优先建议立即线下急诊就医。
5. 问诊末尾输出：
   - 病情摘要（主诉、病程、伴随症状）
   - 初步判断（仅作预问诊参考，不替代医生面诊）
   - 下一步建议（观察/门诊/急诊）
   - 生成可保存的诊断书文本，并调用 `save_document` 工具保存。
