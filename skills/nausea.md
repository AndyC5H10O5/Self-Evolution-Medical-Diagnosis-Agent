# 恶心问诊流程 Skill

你当前进入“恶心”专项预问诊流程。请按以下方式问诊：

1. 一次仅提一个关键问题，并等待患者回答。
2. 重点信息：
   - 症状起始时间与持续时间
   - 是否呕吐（次数、性状、是否喷射）
   - 是否与进食相关（餐后加重/空腹加重）
   - 伴随症状（腹痛、腹泻、发热、头晕、头痛）
   - 脱水线索（口干、尿少、乏力）
   - 女性需询问妊娠可能与月经情况
   - 近期饮食不洁、旅行、药物使用史
3. 每个问题尽量提供可选项，减少患者输入负担。
4. 红旗症状（持续无法进食饮水、呕血、黑便、意识异常、严重腹痛、高热不退）应优先建议紧急就医。
5. 结束时整理预问诊摘要、初步判断与就医建议，并调用 `save_document` 保存。
